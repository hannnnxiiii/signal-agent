# GitHub Trending 日报 - 2026-03-14

## 概览
今日 GitHub Trending 继续聚焦 AI Agent 生态。从 Claude 官方插件目录到多平台聊天机器人基础设施，再到具备学习能力的 Agent 记忆系统，开发者正在围绕 Agent 构建完整的工具链。同时，端侧 AI 部署和高效推理框架也保持活跃。

## 精选仓库
### 1. [anthropics/claude-plugins-official](https://github.com/anthropics/claude-plugins-official)
- 分类：agent
- 简介：Anthropic 官方维护的 Claude Code 高质量插件目录，基于 MCP 协议构建。
- 值得关注：随着 Claude Code 用户增长，官方插件生态逐渐成型。这个仓库为开发者提供了经过审核的插件集合，涵盖各类开发场景，是构建 Claude Code 工作流的重要入口。

### 2. [AstrBotDevs/AstrBot](https://github.com/AstrBotDevs/AstrBot)
- 分类：agent
- 简介：Agentic IM 聊天机器人基础设施，支持 QQ、Telegram、Discord 等多平台，集成 OpenAI、Gemini、Llama 等多种 LLM。
- 值得关注：在即时通讯场景落地 AI Agent 仍面临平台适配、多模型切换等工程挑战。AstrBot 提供了开箱即用的解决方案，并支持 MCP 协议，降低了在聊天场景中部署 Agent 的门槛。

### 3. [vectorize-io/hindsight](https://github.com/vectorize-io/hindsight)
- 分类：agent
- 简介：具备学习能力的 Agent 记忆系统，让 Agent 能够从交互中不断积累知识。
- 值得关注：当前大多数 Agent 缺乏有效的长期记忆机制，每次对话都是全新的上下文。Hindsight 尝试解决这个问题，让 Agent 真正具备「学习」能力，是实现更智能 Agent 的关键拼图。

### 4. [microsoft/BitNet](https://github.com/microsoft/BitNet)
- 分类：agent
- 简介：微软官方推出的 1-bit LLM 推理框架，实现极致压缩的大模型推理。
- 值得关注：1-bit 量化技术能将模型体积压缩至传统格式的 1/32，同时保持可用精度。这对端侧部署和降低推理成本意义重大，是 AI 基础设施层的重要突破。

### 5. [google-ai-edge/LiteRT](https://github.com/google-ai-edge/LiteRT)
- 分类：full-stack
- 简介：TensorFlow Lite 的继任者，Google 推出的端侧高性能 ML 与 GenAI 部署框架。
- 值得关注：端侧 AI 正在从实验走向生产。LiteRT 针对边缘设备优化了模型转换、运行时和性能，支持手机、嵌入式设备直接运行生成式 AI，是全栈开发者需要关注的基础设施工具。

## 结语
Agent 生态正在快速演进。从底层推理优化（BitNet、LiteRT）到上层应用框架（AstrBot），再到记忆增强（Hindsight）和工具生态（Claude Plugins），完整的 Agent 技术栈正在形成。对于开发者而言，理解这个栈的各层能力，比追逐单个热点更有价值。
